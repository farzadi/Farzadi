## 🚀 Featured Projects

### 🎮 [Hicore — Open Source Game Server](https://hicore.netlify.app)
> *Free, open-source, built for speed*

A high-performance game server designed for **fast-paced real-time multiplayer games**. Built with a custom **RUDP protocol** so you don't have to worry about network delays.

- ✅ Unity3D Client SDK (Unreal Engine coming soon)
- ✅ Matchmaking by rank, level, and custom parameters
- ✅ Social features — friends, rooms, opponents
- ✅ Admin Dashboard — analytics, game control, user progress
- ✅ Minimal setup via Docker
- 📄 MIT Licensed
